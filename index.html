Projects

Here are some projects that I have created through my learning. Each project showcases my skills and dedication to web development.

Feel free to explore the details of each project below:

Caesar Cipher

A simple web application that implements the Caesar cipher encryption and decryption algorithm. It allows users to input text, select a shift value, and see the encrypted or decrypted result instantly.

This project demonstrates my understanding of string manipulation, user input handling, and basic cryptographic concepts. It is built using HTML, CSS, and JavaScript, and features a clean and intuitive user interface.

Calculator

A simple calculator using JavaScript Switch-case logic. This project allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division. It features a user-friendly interface and demonstrates my ability to implement logical operations and handle user inputs effectively.

Clock

A simple clock using the Date class in JavaScript. This project dynamically displays the current time, updating every second to ensure accuracy. It showcases my ability to work with JavaScript's Date object, manipulate the DOM, and implement real-time functionality. The clock features a clean design and serves as a practical example of integrating JavaScript logic into a web application.
